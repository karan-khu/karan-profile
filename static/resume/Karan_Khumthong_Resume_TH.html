  • Platform & infrastructure: ย้าย service จาก PM2 → Docker → self-managed Kubernetes ด้วย Helm + ArgoCD GitOps; แปลง batch script เดิมให้กลายเป็น Kubernetes CronJob
  • Collaboration: ทำงานร่วมกับ stakeholder ที่ไม่ใช่สาย tech (operations, sales) เพื่อเก็บ requirement; วางแผน release ทั้ง web, mobile และ backend

ผลงานฝั่งทีม tech: ออกแบบ frontend boilerplate มาตรฐาน (Vite + Preact + TS + antd + react-query + zustand + react-hook-form + yup + Vitest + Husky) — ใช้เป็น starting point มาตรฐานสำหรับ web project ใหม่

ผลงานก่อนหน้าในตำแหน่งเดียวกัน: frontend ของระบบ CRM (หน้าจอ order management และ sales order ร่วมเก็บ requirement และรับ change request), portal ฝึกอบรมสำหรับองค์กรพันธมิตรภายนอก และโมดูล attendance & HR บน mobile app (เช็คอิน / ออก, ขอเบิกค่าใช้จ่าย, ขอลางาน)
